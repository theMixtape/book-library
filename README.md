# 📚 Book Library

Консольний застосунок для керування особистою колекцією книг на Python із збереженням даних у форматі JSON.

---

## 🛠️ Технології

- **Python** — основна мова розробки
- **JSON** — зберігання даних

---

## ⚙️ Функції

- ➕ **Додавання книги** — назва, автор, рік видання
- 📋 **Перегляд колекції** — повний список наявних книг
- ❌ **Видалення книги** — прибрати запис із бібліотеки
- 💾 **Автозбереження** — кожна зміна одразу зберігається у файл

---



## 📁 Структура проєкту

```
book-library/
│
├── library.py       # Основний файл програми
└── library.json     # База даних книг (створюється автоматично)
```
